app.request()

app.request() is used to perform Ajax-requests to the server.
The server returns the following type json

code example

var myCallbackFunc = function(success, result) {
	if ( success === true ) {
		// do something with result
	}
};
var postParams = {
	someVar: 'someVal',
	anotherVar: 123
};
app.request( postParams, myCallbackFunc );

response example

{
	"success" : BOOLEAN,      // application-level status
	"data"    : MIXED,        // operated data
	"error"   : STRING|OBJECT // if application-level error, instead of 'data'
}

App.Request will catch following error-levels: server and application. You do not need to cantch errors.

WARNING !

App.Request depends App.Alert !
